# Phần Mềm Quản Lý Sinh Viên 

Dự án phát triển ứng dụng quản lý sinh viên sử dụng ngôn ngữ **C#**, giao diện **Windows Forms** và hệ quản trị cơ sở dữ liệu **SQL Server**. 

##  Tổng Quan Dự Án
Hệ thống hỗ trợ quản lý thông tin sinh viên, giảng viên và điểm số một cách khoa học, giúp số hóa quy trình quản lý tại các cơ sở đào tạo.

## 🛠 Công Nghệ Sử Dụng
* **Ngôn ngữ:** C# (C-Sharp)
* **Framework:** .NET Framework / WinForms
* **Database:** SQL Server (SSMS 22)
* **Công cụ:** Visual Studio 2022

## 🚀 Hướng Dẫn Cài Đặt

### 1. Yêu cầu hệ thống
* Đã cài đặt **SQL Server Management Studio (SSMS)**.
* Đã cài đặt **Visual Studio** .

### 2. Thiết lập Cơ sở dữ liệu
1. Mở SSMS và kết nối với Server của bạn.
2. Tạo mới một Database có tên (ví dụ: `QuanLySinhVien`).
3. Chạy file script SQL
4. tạo một số đối tượng

### 3. Kết nối và Chạy ứng dụng
1. Mở file giải pháp `.sln` bằng Visual Studio.
2. Kiểm tra lại chuỗi kết nối (`Connection String`) trong file cấu hình để đảm bảo trỏ đúng về SQL Server cục bộ của bạn.
3. Nhấn **F5** hoặc nút **Start** để khởi chạy ứng dụng.
